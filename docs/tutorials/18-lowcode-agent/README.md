## 平台简介

### Coze（扣子）
由字节跳动推出的 AI Bot 开发平台，完全云端、可视化操作，适合快速搭建面向 C 端的 AI 助手。

### Dify
开源的低代码 AI 应用开发平台，支持本地部署，适合需要数据隐私和深度定制的场景。
