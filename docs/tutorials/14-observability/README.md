# 第 14 章: 可观测性、故障排查与降级兜底

## 本章目标

掌握 Agent 运行状态的观测手段，学会排查 Agent 常见故障并设计降级方案。

## 章节内容

- [ ] Tracing：记录 Agent 每一步的思考与行动
- [ ] 常见故障排查：死循环、上下文溢出、工具调用失败、输出格式错误
- [ ] 分级降级策略：模型降级、工具降级、人工兜底
- [ ] Agent 幻觉的三种治理手段：Prompt 层 · 工具层 · 校验层
- [ ] 面试高频问法
