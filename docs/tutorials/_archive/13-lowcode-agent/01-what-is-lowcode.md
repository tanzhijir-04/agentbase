# 01 - 什么是低代码 Agent 平台？

> 用大白话讲清楚"低代码 Agent"到底是什么，为什么你需要它

---

## 先说结论

**低代码 Agent 平台** = 一个可视化的网站/工具，让你**不用写代码**就能搭建自己的 AI 助手（Bot）。

就像 Wix/Squarespace 让你拖拽建网站一样，Coze、Dify 这些平台让你拖拽建 AI Agent。

---

## 没有平台的时代

想象你想做一个 AI 客服机器人，传统做法：

```
1. 学 Python/Node.js
2. 学 LLM API 调用（OpenAI、Claude 等）
3. 搞懂 Prompt Engineering
4. 自己处理对话历史、记忆
5. 自己写工具调用（查数据库、调 API）
6. 部署上线、运维
```

这个过程，**一个有经验的开发者也需要几周到一个月**。

---

## 低代码平台做了什么？

这些平台把上面 6 步**全部封装成可视化操作**：

### 1. 可视化 Bot 搭建
不需要写代码，像搭积木一样组合功能模块。

### 2. 内置 LLM 调用
平台帮你管理 API Key、模型切换、上下文窗口，你只需要选模型和写 Prompt。

### 3. 知识库
上传文档（PDF、Word、网页），平台自动做向量化、检索，Agent 自动查。

### 4. 插件系统
内置搜索引擎、图片生成、代码执行等工具，一键开启。

### 5. 工作流（Workflow）
拖拽连线就能定义"用户问问题 → 搜索知识库 → 调用 API → 生成回复"这样的流程。

### 6. 一键发布
Bot 做好后直接发布到微信、飞书、网页等渠道。

## 主流平台对比

| 平台 | 公司 | 特点 | 适合场景 |
|------|------|------|----------|
| **Coze**（扣子） | 字节跳动 | 完全云端、中文友好、插件丰富 | 快速搭建面向 C 端的 AI 助手 |
| **Dify** | 开源社区 | 开源可自部署、数据隐私、深度定制 | 企业内部应用、需要数据隔离 |
| **GPTs** | OpenAI | 和 ChatGPT 深度集成 | ChatGPT Plus 用户 |
| **百度智能云千帆** | 百度 | 企业级、文心大模型 | 百度云生态用户 |

---

## 你能做什么？

用这些平台，**零基础用户 30 分钟**就能做出：

- 客服机器人（自动回答常见问题）
- 文档问答助手（上传 PDF 让 AI 帮你查）
- 内容生成工具（自动写文案、摘要）
- 翻译助手
- 学习助手（从资料中提取知识点）
