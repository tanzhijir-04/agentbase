# 02 - Coze 入门：创建你的第一个 Bot

> 20 分钟，零代码，从注册到发布一个能回答问题的 AI 助手

---

## 准备工作

只需要两样东西：
1. **一个手机号**（注册 Coze 用）
2. **一个浏览器**（Chrome/Edge 都可以）

不需要信用卡，不需要 API Key。

---

## 第一步：注册 Coze

1. 打开 **[coze.cn](https://www.coze.cn)**（国内版）或 **[coze.com](https://www.coze.com)**（国际版）
2. 用手机号注册登录
3. 登录后你会看到控制台首页

---

## 第二步：创建 Bot

1. 点击左侧 **"Bot"** 标签
2. 点击右上角 **"创建 Bot"** 按钮
3. 输入 Bot 名称：比如 `学习助手`
3. 输入功能介绍：比如 `回答学习相关的问题`

---

## 第三步：写 Prompt（最重要）

Prompt 就是告诉 AI"你要怎么做"的指令。不需要复杂，直接说人话：

```
你是一个学习助手。当用户问你问题时：
1. 先理解问题
2. 如果问题涉及特定知识点，先解释概念
3. 再举例说明
4. 最后问用户是否理解了

回复要通俗易懂，多用例子和比喻。
```

**小技巧**：开头用"你是一个…"来定义角色，AI 会更听话。

---

## 第四步：加知识库

> 知识库让 Bot 能回答你私有的内容（你的文档、产品手册等）

1. 在 Bot 编辑页面找到 **"知识"** 选项
2. 点击 **"添加知识"**
3. 选择 **"文本"** 类型
4. 上传一个文本文件（比如你的产品介绍 PDF）
5. 等几秒钟让系统处理

现在你的 Bot 就能根据上传的文档回答问题了。

---

## 第五步：加插件

> 插件让 Bot 能做"事情"——比如联网搜索、生成图片

1. 在 Bot 编辑页找到 **"插件"** 选项
2. 点击 **"添加插件"**
3. 看看插件市场，找到你想加的功能：
   - **必应搜索**：让 Bot 能联网查最新信息
   - **AI 绘画**：让 Bot 能根据描述生成图片
   - **计算器**：处理数学计算
4. 点击启用即可

---

## 第六步：调试

右侧有个**预览窗口**，你可以直接打字测试：

```
你：什么是 Agent？
Bot：Agent 就是能自主完成任务的 AI 程序...
你：帮我搜索 2024 年诺贝尔奖得主
Bot：（联网搜索后给出答案）
```

不满意？返回修改 Prompt，再试，直到满意为止。

---

## 第七步：发布

1. 点击右上角 **"发布"**
2. 选择发布渠道：
   - **网页**：生成一个分享链接
   - **微信/飞书**：需要绑定对应账号
   - **API**：给开发者用
3. 点击确认，你的第一个 Bot 就上线了！

---

## 常见问题

**Q: 为什么 Bot 回答不准确？**
A: 通常是 Prompt 太简单。试着给它更具体的指令："当用户问 X 时，先做 A 再做 B"。

**Q: 知识库上传后要多久生效？**
A: 一般几秒钟到一分钟，大文件可能需要更久。

**Q: 免费版有什么限制？**
A: Coze 免费版每日有消息条数限制，但对于个人学习完全够用。
