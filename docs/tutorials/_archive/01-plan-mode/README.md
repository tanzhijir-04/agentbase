## 🎯 核心概念

Plan Mode让Agent在执行复杂任务前：
1. **分析用户意图** - 理解用户想要什么
2. **生成执行计划** - 将任务分解成步骤
3. **展示给用户确认** - 用户可以修改计划
4. **按计划逐步执行** - 可追踪进度
