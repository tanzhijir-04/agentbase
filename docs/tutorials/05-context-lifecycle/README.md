## 🎯 核心概念

### 为什么需要？

当对话历史过长时，会面临以下问题：
1. **超出上下文窗口**：LLM有固定token限制（如GPT-4的128K tokens）
2. **性能下降**：上下文越长，推理越慢
3. **成本增加**：token数量直接影响API费用

### 五种主要压缩技术

| 技术 | 原理 | 优点 | 缺点 |
|------|------|------|------|
| 滑动窗口 | 只保留最近N条 | 实现简单 | 可能丢失早期信息 |
| 摘要压缩 | LLM生成摘要 | 保留关键信息 | 需要额外调用 |
| 关键信息提取 | 提取实体/决策/意图 | 高度结构化 | 可能丢失细节 |
| 重要性评分 | 按重要性过滤 | 智能保留 | 需设计评分算法 |
| 分层存储 | 多层级存储 | 平衡效率 | 实现复杂 |
