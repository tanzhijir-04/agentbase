# 第 11 章: 安全边界与风险治理

> 当 Agent 能自主调用工具时，安全问题就不再是可选项了。

## 本章目标

理解 Agent 面临的安全威胁模型，掌握沙箱隔离、Prompt 注入防护、权限控制等核心安全机制。

## 前置知识

- 第 09 章：Skills/Plugins（有了工具扩展才需要考虑安全）

## 章节内容

### 1. 沙箱隔离（原 08-sandbox 内容）

- 进程隔离
- 文件系统隔离（工作目录锁定）
- 网络访问控制

### 2. Prompt 注入攻击与防护

- 什么是 Prompt 注入？直接注入 vs 间接注入
- 防护策略：输入过滤、指令隔离、最小权限原则
- 分级防御模型

### 3. 数据泄露与隐私

- Agent 在思考过程中可能暴露敏感数据
- 工具调用中的参数泄露风险
- 对话历史的隐私清理

### 4. 工具调用权限控制

- 工具白名单机制
- 参数校验与注入检测
- 调用频次限制

### 5. 内容合规校验

- 输出内容过滤
- 敏感操作二次确认
- 审计日志
