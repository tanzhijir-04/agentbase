# JSON Format — JSON格式化

格式化、验证和美化JSON数据。

## 功能

- JSON格式化（缩进美化）
- JSON验证（检查语法错误）
- JSON压缩（去除空格）

## 示例

用户输入：{"name":"test","value":123}
输出：
  {
    "name": "test",
    "value": 123
  }

## 注意

- 发现语法错误时指出具体位置
- 默认缩进为2个空格
