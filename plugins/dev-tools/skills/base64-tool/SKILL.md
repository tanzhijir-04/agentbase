# Base64 Tool — Base64编解码

对文本进行Base64编码或解码。

## 功能

- 文本 -> Base64编码
- Base64 -> 文本解码

## 示例

用户："base64编码 hello"
输出：编码结果

用户："base64解码 aGVsbG8="
输出：解码结果

## 注意

- 编码使用UTF-8字符集
- 无效的Base64输入提示错误
