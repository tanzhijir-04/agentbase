# Case Converter — 大小写转换

将用户提供的文本转换为大写或小写。

## 功能

- 全部转为大写（UPPER CASE）
- 全部转为小写（lower case）
- 首字母大写（Title Case）

## 示例

用户："转大写 hello world"
输出："HELLO WORLD"

用户："转小写 HELLO WORLD"
输出："hello world"

## 注意

- 确认用户希望的是全大写/全小写还是首字母大写
- 保留原始文本中的标点和格式
